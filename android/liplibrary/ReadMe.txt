Test Cases Execution Steps:
1. Open Run menu -> Edit Configurations
2. Add a new Android Tests configuration
3. Choose a module
4. Add a specific instrumentation runner:
   android.support.test.runner.AndroidJUnitRunner
5. Save Configuration
6. Select Module and run Configuration

Test Cases Execution Steps using Gradle command:
1. Launch emulator or connect device to android studio
2. In command line execute connectedCheck or connectedAndroidTest
